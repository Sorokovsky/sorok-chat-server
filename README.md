# SorokChatServer - Сервер українського месенджера

Сервер побудовано на платформі ASP.NET Web API в рамках виконання лабораторної по прикладній криптології.

Функціонал сервера:

1. реєстрація, вхід та вихід користувачів із системи - автентифікація в системі;
2. створення та менеджмент чатів;
3. можливість написати повідомлення до чату;
4. обмін ключами між сервером та клієнтом для криптографічних алгоритмів;

## Автентифікація користувача

Користувач при реєстрації вказує наступні дані: електронна адреса, пароль, ім'я, прізвище, по батькові.
При створені нового користувача автоматично додаються такі поля: унікальний ідентифікатор, дата створення та дата
оновлення профілю. Ці поля є базовими для усіх сутностей у базі даних.

Для входу в систему потрібні такі дані: електрона адреса.

Далі створюється пара access-token та refresh-token

## Створення та менеджмент чатів

У наявній версії сервера, можна лише створити чат та додати/видалити користувача, а також написати повідомлення до чату.

## Повідомлення

Для того, щоб оповістити активних користувачів, що є нове повідомлення, було використано websocket.

У ASP.NET для цих цілей було використано вбудовану бібліотеку SignalR.

## Обмін ключами для криптографії

Для обміну ключами, в рамках лабораторної роботи було використано протокол 3-DH(тричі Діффі-Хеллмана).
